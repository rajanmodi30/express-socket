
# Socket Express

A socket io express server with redis adapters to handle the socket from anywhere with adapters from different server that will directly be handled with redis


## Installation

Install on development

```bash
  yarn Install

  yarn dev
```

## Installation

Install on development

```bash
  yarn build

  yarn start
```
